# PAI-Programs

PAI-Programs is a collection of Python implementations of core **Artificial Intelligence (AI)** concepts and **search algorithms**, developed as part of learning and practicing **Principles of Artificial Intelligence (PAI)**.  
The repository focuses on clarity, correctness, and educational value, making it suitable for students, beginners, and instructors.
